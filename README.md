## 🌍 Overview

Pathik is a **community-first platform** designed to connect individuals in need with **NGOs, volunteers, and crisis support services** through a unified, easy-to-use digital system.

---

## 🌍 Problem Statement

Many individuals, especially in emergency or vulnerable situations, struggle to quickly access:

* Nearby NGO support and relief assistance
* Food and shelter services
* Volunteer help during crises
* Safe navigation during emergencies

Existing solutions are fragmented and difficult to use when help is needed most.

**Pathik bridges this gap** by offering a unified digital platform for rapid community coordination and assistance.

---

## 💡 Solution

Pathik empowers communities through:

* **🚨 Crisis Reporting** – Quickly report emergencies with location and details
* **🤝 Volunteer Onboarding** – Join as volunteer, NGO member, or support coordinator
* **💬 Smart Assistant** – Guided support for reporting, volunteering, and finding assistance
* **📱 Responsive Design** – Works seamlessly on mobile, tablet, and desktop

## 🚀 Quick Start

### Prerequisites

* Node.js v16+
* npm v8+

### Installation

```bash
# Clone repository
git clone https://github.com/Muditapandey26/Pathik.git
cd Pathik

# Install dependencies
npm install

# Start development server
npm run dev
```

App runs at: `http://localhost:5173`

### Build for Production

```bash
npm run build      # Create optimized build
npm run preview    # Preview production build
npm run lint       # Check code quality
```
